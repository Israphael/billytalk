## Что делать дальше — цикл 2, интерфейс (harness §9)

1. **`core/ipc/`** — именованный канал: имя с `session_id` (только SID —
   коллизия консоль/RDP), **DACL обязателен** (по умолчанию Windows даёт чтение
   «Всем», а по каналу идут расшифровки), `FILE_FLAG_FIRST_PIPE_INSTANCE`,
   `PIPE_REJECT_REMOTE_CLIENTS`, рукопожатие `hello`/`hello_ack`.
2. **`ui/` каркас** — второй процесс, wxPython; `ui/` не импортирует `ctypes`
   ни для чего, кроме стилей окна.
3. **Трей на ctypes в ядре** — `NOTIFYICON_VERSION_4` + `NIM_SETVERSION`,
   идентификация `hWnd`+`uID` (не GUID), `NIF_SHOWTIP`, окно **настоящее верхнего
   уровня** (не message-only — `TaskbarCreated` туда не приходит), обработчик
   идемпотентен. Шесть состояний значка, включая «офлайн» (спека §3).
4. **Плашка** — `WS_EX_NOACTIVATE|TRANSPARENT|LAYERED|TOOLWINDOW|TOPMOST`,
   показ только `SW_SHOWNOACTIVATE`; живёт через `Finalizing`/`Delivering`;
   нельзя отключить, пока пользователь ни разу не использовал `Ctrl+Alt+Z`.
5. **`insert/verify.py` — на UIA из S2**: `TextPattern` после позиции каретки;
   исходы: найден → `inserted`; элемент недоступен/паттерна нет →
   `verify_impossible`, **молча**; значение не изменилось → `paste_failed`,
   громко. UIA **только чтение**; `ValuePattern.SetValue` — никогда (единственное
   исключение — пустое незащищённое поле). Флаги исключения буфера — только для
   процессов из списка достоверной проверки.
6. **Ctrl+Alt+Z / Ctrl+Alt+X** — «последнее» = свежайшее из показанных;
   `retry_count>0` исключается; старше 2 минут — подтверждение; в терминал —
   подтверждение всегда.
7. Окна настроек и истории (**вставку из истории исполняет ядро** — у него HWND
   и общий буферный мьютекс), захват хоткея (таймаут 30 с, снимается при разрыве
   канала), ранжированный микрофон + `WM_DEVICECHANGE` + перезагрузка PortAudio
   **только при остановленном потоке**, тумблер и запасная привязка,
   `WM_POWERBROADCAST` / `WM_QUERYENDSESSION` (уже есть события машины `Suspend`
   и `Exit` — подключить к настоящим сообщениям окна).

Готово, когда: значок жив после убийства проводника, плашка не крадёт фокус,
история ищет/вставляет/экспортирует, verify отличает `inserted` от
`verify_impossible`, захват хоткея не оставляет кнопку подавленной при падении
интерфейса, тесты зелёные.

## Измерено циклом 1 — не переоткрывай и не забывай

- **Сигнал вставки существует**: UIA `TextPattern`, появление ~152 мс (медиана);
  `WM_GETTEXT` мёртв везде (урок S7). `RESTORE_DELAY_MS=300` подтверждён измерением.
- **Фоновая доставка в новый Блокнот: ноль из трёх маршрутов** — и ноль краж
  фокуса. Ступень «б» выключена по умолчанию; `_WwG` в чёрном списке навсегда.
- **Прогретый пул обязателен**: 350 против 590 мс. Мёртвый reused keep-alive
  получает один тихий повтор **только на фабрично-свежем** соединении.
- `MOUSEINPUT.mouseData` для инжекта XBUTTON — значение **без сдвига**; HIWORD —
  формат хук-структуры. Со сдвигом `SendInput` вернёт успех, события не будет.
- pywin32 `CredWrite` принимает только `str` (сам кодирует UTF-16-LE); чтение
  отдаёт байты. Кривой блоб → `SecretUndecodable`, никогда не сырой
  `UnicodeDecodeError` (в нём ключ целиком).
- Оверфлоу аудио-колбэка — **не** `MicError`: то событие финализирует запись.
- Бюджет промпта: **420 символов кириллицы / 500 латиницы** — 492 кириллических
  уже 239 токенов, проверено настоящим токенизатором Whisper.
- Пять багов спеки нашлись реализацией, одиннадцать дефектов — адверсариальным
  ревью. Показатель «реализация находит ошибки» держится: пиши в OPEN-QUESTIONS.

## Как здесь принято работать

- **Измеряй, не оценивай.** Сравнения чередованием, минимум восемь пар.
- **Не угадывай молча.** Неоднозначность → реализуй лучшее чтение, оба варианта —
  в `OPEN-QUESTIONS.md`.
- **Тест, зелёный до починки, ничего не стоит.** Ломай починку, смотри на красный.
- **Веди документацию параллельно.** Репозиторий публичный и портфолийный;
  отменённое решение записывается, а не затирается.
- **Никогда не логируй** аудио, текст расшифровки и любую его подстроку, заголовок
  окна, нажатия сверх хоткея. Механизм — `core/logging_setup.py`; тип `Sensitive`
  возвращает константу, не исключение.
- **Спрашивай заказчика только там, где нужен его выбор.**
